### 常用系统工具安装

* [*] 安装 brew 工具，参考 http://brew.sh
* 编辑器，推荐 Vim/Emacs/Sublime Text 3
* Terminal: iTerm
* shell: oh-my-zsh
* 用 brew 安装 git, python
* 用 pip 安装 virtualenv，并使用 virtualenv 配置虚拟环境
