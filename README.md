### 更新系统

### 安装 Xcode
* Xcode
* Xcode commandline tools

### 常用系统工具安装
带*号的是必须安装，其他推荐

* [*] 安装 brew 工具，参考 http://brew.sh
* 编辑器，推荐 Vim/Emacs/Sublime Text 3
* Terminal: iTerm
* shell: oh-my-zsh
* [*] 用 brew 安装 git, python
* [*]用 pip 安装 virtualenv
